# Weather API Project 🌤️

Projet d'apprentissage pour comprendre les APIs avec Python.

## Objectifs

1. ✅ Faire une requête HTTP GET
2. ✅ Parser du JSON
3. ✅ Gérer les erreurs
4. ✅ Utiliser une API key
5. ✅ Comprendre le rate limiting

## Étapes

### Étape 1 : Requête simple (ce qu'on va faire maintenant)
- Appeler l'API Open-Meteo (gratuite, pas besoin de clé !)
- Afficher la météo actuelle

### Étape 2 : Ajouter des fonctionnalités
- Historique des requêtes
- Sauvegarde dans un fichier JSON
- Alertes météo

### Étape 3 : Aller plus loin
- API avec authentification (OpenWeatherMap)
- Créer ta propre API avec FastAPI
- Dashboard web simple

## Prérequis

```bash
pip install requests
```

## Lancer le projet

```bash
python weather.py
```
